# Edge-o-Matic 3000 Mock Server
This mocks a Websocket server for the Edge-o-Matic 3000.

It's clearly a work in progress.

## Deps.

`npm install`

## Node.

`node server.js`

## Docker.

`docker build -t eom-mock-server .`

`docker run -p 8080:8080 eom-mock-server`

## Compose.

`docker-compose up`
